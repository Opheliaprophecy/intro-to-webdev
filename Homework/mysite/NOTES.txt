
## General Keyboard stuff

* ⌘ + space = search / launch program
* ⌘ + left = home
* ⌘ + right = end
* ⌘ + shift + 4 = screenshot
* ctrl + scroll (2 fingers up/down) = ZOOOM!
* move mouse top left corner = sleepy night-night

## VS Code Keyboard stuff

* ⌘ + P = quick file switcher (press twice to quickly go to last file)
* ⌘ + D = select multiple (highlight something then press it to select others)
* option + up/down = move line up / down
* option + shift + up/down = duplicate line up/down

## Start server

To start the server run: `browser-sync start --server --files .`
(you should be able to type brow [tab] then just press up once)
